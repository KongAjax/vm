## Button

### 代码演示

```html
<vm-button></vm-button>
```  

### API
属性 | 说明 | 类型 | 默认值
-----|-----|-------|------
type | 按钮类型，可选值为primary/large/warning或者不设 | string | primary
size | 按钮大小，可选值为large、small | string | normal
radius | 设置圆角值 | string、number | normal
hollow | 是否空心 | boolean | false
disabled | 设置禁用 | boolean | false
