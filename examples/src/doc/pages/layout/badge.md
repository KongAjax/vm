## Badge

### 代码演示
```html
<vm-badge>10</vm-badge>
```  
### Slot
名称 | 说明 | 默认值
-----|-----|-------|------
- | 如未设置props text属性 |


### Props
属性 | 说明 | 类型 | 默认值
-----|-----|-------|------
color | 设置徽章颜色，可设置vmui公共配置 | String | badge.color
text | content | Number, String | -

### 全局配置
Vmui.config => badge.color
