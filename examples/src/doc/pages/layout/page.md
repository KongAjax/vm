## Page

### 代码演示

```html
<vm-page>
    <template slot="header">
        <div>header</div>
    </template>
    页面内容主体
    <template slot="footer">
        <div>footer</div>
    </template>
</vm-page>
```  

### Slot
名称 | 说明 | Slot-Scpoe
-----|-----|-------|------
header | 三层布局顶部位置 | -
footer | 三层布局底部位置 | -
